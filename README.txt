 ABOUT BOOKSTORE
---------------

The Bookstore is a design concept project for week 1 at epicodus.
Its is a practice in using features, views, native and custom content to create a basic bookstore page.

INSTALLATION
-------------
1. Go to http://localhost:8888/MAMP/ and under the "Tools" menu, select "phpMyAdmin"

2. Please select the from here if the DataBase for 'bookstore' is not present then please import the database.
  Importing the database: the database can be found under bookstore/sites/db-backup/bookstore.sql.zip

3. from http://localhost:8888/MAMP/?language=English select 'my website' from the menu bar, which will navigate you to the Bookstore.

SPECS
-----

An auth user can add bookreviews.
user: clicks add bookreview link.
Output: a form appears for a user to enter content for a reviewed book.

All auth users can view a special coupon.
user: logged in
Output: can see a coupon for 25% off all Sci/fi books.
